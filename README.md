Этот бот сохраняет до 10ти, присланых пользователем, изображений в папке пользователя на сервере.

Способен:
- Выводить список сохранённых изображений
- Отправлять изображения пользователю обратно
- Удалять сохранённые изображения пользователя с сервера.

Рабочий пример: @image_keeper_bot

Известные проблемы:
- Пока не может адекватно обработать добавление сразу нескольких фото. Обрабатывает последнее добавленное.
